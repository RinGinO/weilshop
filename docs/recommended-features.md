# Рекомендуемый пользовательский функционал WeilShop

Этот документ фиксирует функции, которые повышают удобство выбора автохимии и делают сайт полезнее для пользователя.

## 1. MVP

Функции, которые стоит заложить в первую очередь:

- `Подбор по задачам ухода`
  Пользователь выбирает не товар, а задачу: мойка кузова, очистка салона, удаление битума, уход за пластиком.

- `Пошаговые комплекты товаров`
  На странице задачи ухода и в карточке товара показываются рекомендуемые товары по шагам: шаг 1, шаг 2, шаг 3.

- `FAQ и инструкции в карточке товара`
  Короткие и практические ответы на вопросы по использованию конкретного средства.

- `История статусов заявки`
  Пользователь видит не только текущий статус, но и историю обработки заявки.

- `Уведомления о статусе заявки`
  На первом этапе по `email`, позже можно добавить `SMS` и `Telegram`.

- `Избранное`
  Пользователь может сохранить товары и вернуться к ним позже.

- `История просмотров`
  Удобно для возврата к недавно просмотренным товарам.

- `Консультация по подбору`
  Кнопка для связи с администратором или менеджером по подбору средств.

## 2. Post-MVP

Функции, которые сильно улучшают опыт и увеличивают конверсию:

- `Сравнение товаров`
  Сравнение по объему, назначению, способу применения и уровню пользователя.

- `Мини-квиз для подбора автохимии`
  Подбор по нескольким вопросам: что нужно очистить, какой тип загрязнения, какой нужен результат.

- `Связанные товары и аналоги`
  Блоки "с этим товаром берут", "следующий шаг ухода", "похожие товары".

- `Отзывы с опытом применения`
  Отзывы с акцентом на практический результат, а не только общую оценку.

- `Фильтр по уровню пользователя`
  Например: новичок, опытный пользователь, домашний уход.

- `Повтор заявки`
  Быстрое повторное формирование заявки на основе прошлой.

## 3. Future Features

Функции, которые особенно хорошо подойдут на следующих этапах:

- `Подбор сезонных наборов`
  Например: зимний уход, весенняя мойка, подготовка к дальней поездке.

- `Персональные рекомендации`
  На основе просмотренных товаров, избранного и прошлых заявок.

- `Push-уведомления в мобильном приложении`
  После запуска Android-приложения.

- `Сканер QR или SKU`
  Для быстрого открытия карточки товара в приложении.

## 4. Особенности именно для автохимии

Для этой ниши особенно важно предусмотреть:

- предупреждения о совместимости поверхностей
- меры предосторожности
- рекомендации по последовательности использования средств
- понятные инструкции для новичков
- акцент на сценарий применения, а не только на список характеристик
