# Роли и права администраторов WeilShop

## Базовые роли

### `Super Admin`
Полный доступ ко всем разделам системы:
- создание новых администраторов
- назначение ролей
- блокировка администраторов
- системные настройки
- полный доступ к каталогу, заявкам и контенту

### `Catalog Manager`
Отвечает за товарную часть:
- создание и редактирование товаров
- управление категориями
- управление брендами
- управление задачами ухода
- настройка связей товаров с задачами ухода

### `Content Manager`
Отвечает за контент:
- статьи
- инструкции
- FAQ
- SEO-метаданные
- контент страниц брендов

### `Request Manager`
Отвечает за обработку заявок:
- просмотр списка заявок
- просмотр карточки заявки
- изменение статуса заявки
- добавление внутреннего комментария
- просмотр истории изменения статусов

### `Consultation Manager`
Отвечает за консультации по подбору:
- просмотр списка консультаций
- просмотр карточки консультации
- назначение ответственного
- подготовка ответа пользователю
- изменение статуса консультации

### `SEO / Merch Manager`
Отвечает за коммерческую подачу и поисковую видимость:
- настройка SEO-метаданных
- управление подборками и комплектами товаров
- настройка приоритетов вывода контента и товаров

## Рекомендации по правам

- один администратор может иметь несколько ролей
- критические действия должны логироваться
- создание новых администраторов доступно только `Super Admin`
- изменение ролей доступно только `Super Admin`
- обработку заявок и консультаций лучше разделять по ролям при росте проекта
