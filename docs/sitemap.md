# Sitemap WeilShop

## Публичная часть

- `/` - Главная
- `/catalog` - Каталог
- `/catalog/[category]` - Страница категории
- `/product/[slug]` - Карточка товара
- `/compare/[id]` - Сравнение товаров
- `/tasks` - Список задач ухода
- `/tasks/[slug]` - Страница задачи ухода
- `/tasks/[slug]/kits` - Комплекты по задаче ухода
- `/brands` - Список брендов
- `/brands/[slug]` - Страница бренда
- `/knowledge` - База знаний
- `/knowledge/articles` - Список статей
- `/knowledge/articles/[slug]` - Статья
- `/knowledge/instructions` - Список инструкций
- `/knowledge/instructions/[slug]` - Инструкция
- `/knowledge/faq` - FAQ
- `/about` - О компании
- `/contacts` - Контакты
- `/search` - Поиск
- `/favorites` - Избранное
- `/cart` - Корзина
- `/request` - Оформление заявки
- `/consultation` - Запрос консультации
- `/auth/login` - Вход
- `/auth/register` - Регистрация
- `/auth/forgot-password` - Восстановление пароля

## Личный кабинет

- `/account` - Главная личного кабинета
- `/account/profile` - Профиль пользователя
- `/account/requests` - Список заявок
- `/account/requests/[id]` - Карточка заявки
- `/account/consultations` - Список консультаций
- `/account/consultations/[id]` - Карточка консультации
- `/account/favorites` - Избранное
- `/account/comparisons` - Списки сравнений
- `/account/notifications` - Уведомления
- `/account/history` - История просмотров
- `/account/password` - Смена пароля

## Административная панель

- `/admin` - Точка входа в админку
- `/admin/dashboard` - Дашборд

- `/admin/brands` - Управление брендами
- `/admin/articles` - Управление статьями
- `/admin/instructions` - Управление инструкциями
- `/admin/faq` - Управление FAQ
- `/admin/care-kits` - Управление комплектами товаров
- `/admin/requests` - Управление заявками
- `/admin/consultations` - Управление консультациями
- `/admin/users` - Управление пользователями
- `/admin/admins` - Управление администраторами
