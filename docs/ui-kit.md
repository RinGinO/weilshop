# UI Kit WeilShop

## 1. Основа дизайн-системы

`WeilShop` должен использовать единую UI-систему, чтобы витрина, личный кабинет и админка ощущались как части одного продукта.

Базовые принципы:
- чистый и спокойный интерфейс
- высокая читаемость
- предсказуемые состояния элементов
- акцент на сценарии ухода и практическое использование

## 2. Цветовые токены

## 3. Типографическая шкала

- `Display` — крупные hero-заголовки
- `H1` — ключевые заголовки страниц
- `H2` — заголовки секций
- `H3` — заголовки карточек и подблоков
- `Body L` — длинные описания и вводные тексты
- `Body M` — основной интерфейсный текст
- `Body S` — характеристики и вспомогательный текст
- `Caption` — подписи и служебные элементы

- `sticky action bar` в карточке товара

- `bottom sheet` для быстрых действий
- компактные карточки товаров
