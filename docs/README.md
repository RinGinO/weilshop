# WeilShop Documentation

Этот каталог содержит базовую документацию по архитектуре сайта `WeilShop`.

Состав:
- `architecture.md` - общая архитектура проекта
- `design-concept.md` - рекомендуемая дизайн-концепция сайта
- `ui-kit.md` - базовая структура дизайн-системы и UI-компонентов
- `component-diagram.md` - компонентная диаграмма системы
- `use-case-diagram.md` - use case диаграмма по ролям
- `er-diagram.md` - ER-диаграмма базы данных в формате Mermaid
- `api-contract.md` - черновик REST API
- `prisma-schema.prisma` - расширенная схема данных Prisma под каталог, заявки и пользовательские функции
- `recommended-features.md` - рекомендуемые функции для удобства пользователя
- `sitemap.md` - структура разделов и маршрутов сайта
- `database-schema.md` - логическая схема базы данных
- `admin-roles.md` - роли и права администраторов
- `user-flows.md` - ключевые пользовательские сценарии
- `technical-specification.md` - базовое ТЗ для передачи в разработку
- `implementation-roadmap.md` - дорожная карта реализации
