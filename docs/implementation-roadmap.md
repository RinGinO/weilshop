# Roadmap WeilShop

## Этап 1. Аналитика и проектирование

- финализация структуры страниц
- согласование ролей и прав
- финализация `ER`-модели
- согласование API-контрактов
- подготовка UI-прототипов
- формирование дизайн-системы и UI kit

## Этап 2. Backend foundation

- настройка `NestJS`
- подключение `PostgreSQL`, `Prisma`, `Redis`
- реализация модулей `auth`, `users`, `admins`, `roles`
- базовые миграции
- сиды ролей администратора

## Этап 3. Каталог и контент

- товары, категории, бренды
- задачи ухода
- комплекты товаров по шагам
- статьи, инструкции, FAQ
- загрузка медиафайлов
- SEO-метаданные

## Этап 4. Пользовательская зона

- регистрация и вход
- личный кабинет
- избранное
- корзина
- история просмотров
- сравнение товаров
- уведомления пользователя
- консультации по подбору

## Этап 5. Заявки

- создание заявки из корзины
- история статусов
- кабинет пользователя со статусами
- административная обработка заявок

## Этап 6. Публичная витрина

- главная страница
- каталог
- карточка товара
- страницы задач ухода
- страницы сравнения и консультации
- база знаний
- поиск

## Этап 7. Качество и запуск

- SEO-проверка
- адаптивность
- проверка дизайн-системы на консистентность
- базовая аналитика
- smoke tests
- подготовка production-конфигурации

## Этап 8. Подготовка к Android

- стабилизация API
- выделение мобильных DTO
- подготовка документации для клиента Android
